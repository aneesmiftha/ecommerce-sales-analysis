E-commerce Sales Analysis

Overview

This project analyzes an e-commerce dataset to identify sales trends, top-performing products, and regional performance.

Tools Used

- Excel (Data Cleaning & Initial Analysis)
- SQL (Data Querying & Validation)
- Power BI (Dashboard & Visualization)

Key Insights

- Watches had the highest sales quantity.
- Phones generated the highest revenue.
- Delhi was the top-performing city in terms of revenue.
- Sales peaked in August and were lowest in April.
- Higher revenue in Delhi was driven by higher average order value.

Dashboard

The Power BI dashboard provides insights into:

- Sales by City
- Monthly Sales Trend
- Product Performance

Files in Repository

- ecommerce_dataset.csv

Conclusion

This project demonstrates end-to-end data analysis from raw data to business insights using multiple tools.
